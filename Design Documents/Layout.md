# VR Input Layout

## Special Keys

* R3, L3            => Shift
* R3 + R3, L3 + L3  => Capslock
* RGrip, LGrip      => Space

### Left Controller

* Y => Tab
* X => Special Character Modifier

### Right Controller

* B => Backspace
* A => Enter

---

## Keys

### Left Controller

* L ^   => WHEEL(1,2,3,4,5)
* L <=  => WHEEL(q,w,e,r,t)
* L =>  => WHEEL(a,s,d,f,g)
* L \/  => WHEEL(z,x,c,v,b)
* X     => WHEEL(`,[,\)

### Right Controller

* R ^   => WHEEL(6,7,8,9,0)
* R <=  => WHEEL(h,j,k,l,;)
* R =>  => WHEEL(y,u,i,o,p)
* R \/  => WHEEL(n,m,<,>,?)
* X     => WHEEL(-,=,')
