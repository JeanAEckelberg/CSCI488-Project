# VR Input Layout

## Special Keys

* R3, L3            => CapsLock
* RGrip, LGrip      => Space

### Left Controller

* X => Tab
* Y => Special Character Modifier

### Right Controller

* A => Backspace
* B => Enter

---

## Keys

### Left Controller

* L ^  => WHEEL(q,w,e,r,t)
* L >  => WHEEL(a,s,d,f,g)
* L v  => WHEEL(z,x,c,v,b)
* L <   => WHEEL(1,2,3,4,5)
* X     => WHEEL(`,[,\)

### Right Controller

* R ^  => WHEEL(y,u,i,o,p)
* R <  => WHEEL(h,j,k,l,;)
* R v  => WHEEL(n,m,<,>,?)
* R >   => WHEEL(6,7,8,9,0)
* X     => WHEEL(-,=,')
